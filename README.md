# Pixglyph

Веб-приложение для генерации ASCII-арта из изображений.  
Открой `index.html` — сборка не нужна.

## Как пользоваться

1. Перетащи изображение в левую панель, на холст, или вставь Ctrl+V
2. Выбери стиль
3. Настрой параметры — рендер обновляется автоматически
4. Скачай в нужном формате

## Стили

| Стиль      | Символы                        |
|------------|-------------------------------|
| Matrix     | `. : + % @ # & *`             |
| Lines      | `· - ~ = + T H #`             |
| Cyberpunk  | `· : ; = x X $ █`            |
| Retro      | `. ' \` + ; : , | ! ?`        |
| Block Art  | `░ ▒ ▓ █`                    |
| Halftone   | `. · ° o O 0 @ ■`            |

## Параметры

- **width_chars** — ширина (40–300)
- **contrast / brightness** — коррекция изображения
- **detail_lvl** — low / med / high
- **sharpness** — резкость
- **saturation** — насыщенность (влияет на цветной режим)
- **edge_boost** — усиление контуров (Sobel)
- **char_density** — плотность символов (1–5)
- **invert** — инверсия яркости
- **color_mode** — цвет из оригинального пикселя

## Интерфейс

- **◐ / ◑** — переключение тёмной/светлой темы
- **− / +** — зум ASCII-вывода (5–16px), также Ctrl+колесо на холсте
- **☰** (мобильный) — открыть панель настроек
