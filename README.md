# Pixglyph

Веб-приложение для генерации ASCII-арта из изображений методом дискретизации цвета.

## Запуск

Открой `index.html` в браузере. Сборка не нужна.

## Использование

1. Перетащи изображение в зону загрузки, кликни по ней, или вставь из буфера (Ctrl+V)
2. Выбери стиль
3. Настрой параметры — рендер обновляется автоматически
4. Скачай результат в нужном формате

## Стили

| Стиль      | Описание                                  |
|------------|-------------------------------------------|
| Matrix     | Классические ASCII-символы               |
| Braille    | Unicode Braille, блок 2×4 пикселя        |
| Cyberpunk  | Блочные символы, тёмная палитра           |
| Retro      | Имитация старого терминала               |
| Block Art  | Unicode-блоки ░▒▓█                        |
| Halftone   | Точечный стиль . · ° o O 0 @ ■           |

## Параметры

- **width_chars** — ширина в символах (40–200)
- **contrast** — контрастность (-100 до +100)
- **brightness** — яркость (-100 до +100)
- **detail_lvl** — детализация: low / med / high
- **invert** — инверсия яркости
- **color_mode** — цвет из оригинального пикселя

## Экспорт

- **TXT** — текст UTF-8
- **SVG** — масштабируемый вектор с цветом
- **PNG** — растровый рендер 3× (точное соответствие экрану)
- **HTML** — страница с inline-стилями
- **ANSI** — escape-коды для терминала

## Структура

```
pixglyph/
  index.html
  css/style.css
  js/
    quantize.js    # алгоритм дискретизации
    exporters.js   # экспорт форматов
    app.js         # логика приложения
```
